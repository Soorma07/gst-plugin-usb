Description

GstUsb is a Gstreamer plugin that allows two pipelines communicate via USB link. The plugin consists of two elements to send and receive data respectively:

  • usbsink
  • usbsrc

The philosophy behind GstUsb is to provide agnostic streaming or, in other words, the possibility to stream any kind of multimedia content using these elements.
